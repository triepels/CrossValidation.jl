# Description
Provides simple and lightweight implementations for model validation and hyperparameter tuning in Julia. 

# Features
Validation schemes:
* Model validation
* Hyperparameter tuning
* Nested cross-validation

Resample methods:
* Holdout
* LeavePOut
* KFold

Search methods:
* Exhaustive search

# Installation

```julia
] add https://github.com/triepels/CrossValidation.jl
```
